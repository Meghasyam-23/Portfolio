# Meghasyam's Portfolio

A minimalist, Apple-inspired portfolio website showcasing my projects, skills, and experience as a Computer Science student at Iowa State University.

## Design Philosophy

This portfolio is built with a focus on:

- **Elegant Minimalism**: Clean, sophisticated interface that puts content first
- **Superior Performance**: Optimized code and lightweight animations for smooth experience
- **Thoughtful Interactions**: Subtle animations and transitions enhance usability
- **Accessibility**: Clear typography and navigation for all users

## Features

- **Apple-Inspired Design**: Clean typography, refined color palette, and thoughtful spacing
- **Light/Dark Mode**: Automatically detects system preference with manual toggle
- **Optimized Performance**: Lightweight, vanilla JavaScript without heavy dependencies
- **Unique Visualizations**:
  - Subtle particle network for visual interest
  - Orbital skills visualization that's both informative and elegant

## Technologies Used

- Vanilla HTML, CSS, and JavaScript
- No external dependencies for core functionality
- Responsive design for all device sizes
- System-level font stack for optimal performance

## Sections

1. **Hero** - Clean introduction with subtle animation
2. **Work** - Project showcase with focus on content
3. **About** - Personal background with key areas of focus
4. **Skills** - Innovative orbital skills visualization
5. **Experience** - Clean timeline of professional history and education

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Toggle between light and dark mode with the switch in the header

## Customization

The portfolio uses CSS variables for easy customization:
- Colors are defined in the root selector
- Dark mode colors are separate for perfect contrast in both modes
- Typography, spacing, and animation variables are centralized
